Welcome to the code used to identify, control, and play around with the TQ-CE103 Thermal control apparatus. 
This repo serves as a backup of the data, since to directly control the set-up, you will have to build a .slx file and integrate this in D-Space LabView
This script does not focus on setting up the D-Space connection, instead it leverages an existing D-space connection to control the set-up in python.

| SciPy |
| IPyWidgets |
| Numpy |

| Pandas |
